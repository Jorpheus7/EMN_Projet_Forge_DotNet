[ Install ]

- Install AdventureWorks for SQL Server 2008 (http://msftdbprodsamples.codeplex.com/releases/view/37109)
- Create a SQL server account to access the database and modify the Web.config accordingly
- Execute the *.sql from the folder "_Documentation" in order to :
  - Add a new user (login/password = demo/demo)
  - Add a stored procedure to retrieve the employees' list
  
And that's it !
